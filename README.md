# ML-model-projects
A collection of 9 Machine Learning projects covering supervised, unsupervised, and deep learning. Features predictive models for Diabetes, Rock vs Mine, Fake News, and Spam.  Customer/IMDB Sentiment  MNIST, Fashion MNIST, and Dog vs Cat classification. Built with Python and Scikit-Learn/TensorFlow.

# Machine Learning Project Collection

A comprehensive repository featuring **9 diverse Machine Learning and Deep Learning projects** r

## 🚀 Projects Included

### 🧠 
*   **Diabetes Prediction**: Diagnostic prediction based on medical metrics.
*   **Rock vs. Mine**: Sonar signal classification for underwater objects.
*   **MNIST Digit Classification**: Handwritten digit recognition using Neural Networks.
*   **Fashion MNIST**: Classifying clothing items (shirts, shoes, etc.).
*   **Dog vs. Cat**: Image classification using Convolutional Neural Networks (CNN).
*   **Fake News Detector**: Identifying misinformation in news articles.
*   **Spam Mail Classifier**: Filtering unwanted emails using Text Mining.
*   **Customer Sentiment Analysis**: Analyzing feedback for business insights.
*   **IMDB Review Sentiment**: Movie review polarities (Positive/Negative).

## 🛠️ Tech Stack
*   **Language:** Python
*   **Libraries:** Scikit-Learn, TensorFlow/Keras, Pandas, NumPy, matplotlib,seaborn
*   **Environment:**Jupyter Notebook and Google Colab
